## 开始使用
1. 添加css前置样式
    在使用之前，请在css样式作如下修改：
    ```css
    *,*::before,*::after{margin: 0;padding: 0;box-sizing: border-box}
    ```
2. 安装forlearn-ui
    ```
    npm i --save forlearn-ui-test
    ```
   
3. 引入forlearn-ui
